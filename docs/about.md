# About This Site

Built for CRNA students rotating through cardiac anesthesia.
Content reflects local practice at a single hospital and should not
replace institutional protocols or clinical judgment.

## Disclaimer

This site is an educational reference for anesthesia trainees.
It does not constitute medical advice. Clinical decisions should
be made in consultation with attending physicians and based on
current institutional protocols.
